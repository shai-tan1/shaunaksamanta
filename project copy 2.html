WhiteBoard

A real-time multi-user collaborative whiteboard — draw, annotate, and sync instantly across all connected clients via WebSockets.

RoleFull-stack Developer
StackNext.js, NestJS, Socket.IO, Fabric.js
CodeVisit Site ↗

Overview

WhiteBoard is a multiplayer drawing app where every stroke, shape, and annotation is broadcast to all connected clients in real time. The NestJS backend maintains authoritative canvas state so late-joining users receive the full board — not a blank slate.

The hardest problem was preventing echo loops: when the server rebroadcasts a client's event, that client must not re-apply it. I implemented a client-side event guard keyed on socket ID that filters out self-originated events before they hit the canvas.

What I built

Frontend in Next.js + Fabric.js with pen, highlighter, eraser, shapes, and text tools. Full undo/redo synced across clients. NestJS + Socket.IO backend with room-based broadcasting and state persistence. Presence indicators show who is currently connected.
