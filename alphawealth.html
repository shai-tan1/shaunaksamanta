AlphaWealth

An investment portfolio manager that helps users diversify assets and make smarter allocation decisions.

RoleFull-stack Developer

LiveVisit site ↗

Overview

AlphaWealth is a personal finance tool focused on portfolio diversification. Users can track holdings across asset classes and visualize how their allocations are distributed, helping them spot imbalances and make more deliberate investment decisions.

I wanted to learn TypeScript at scale and Next.js App Router patterns simultaneously, so this project was an intentional deep-dive into both. Tailwind CSS kept the UI consistent without fighting a component library.

What I built

Built the full stack with Next.js — server components handle data-fetching, client components manage interactive charts and forms. Deployed on Vercel with a separate Render backend. TypeScript caught numerous bugs early that would have been silent runtime errors in plain JS.
