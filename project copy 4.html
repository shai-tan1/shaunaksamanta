Overview

Arcade (internally "CRYSTAL") is a deliberate exercise in going back to basics — no React, no build step, no framework overhead. The focus was on building a clean, fast web experience using only platform primitives: vanilla JavaScript, HTML5, and CSS3.

This was a study in keeping a project lean. By stripping away tooling, every performance and UX decision becomes explicit — there is nothing magic happening behind the scenes. The result loads fast and deploys via automated Vercel CI/CD on every push.

What I built

Modular asset organization separating static files from core logic. Adaptive layouts for desktop and mobile. A deep-contrast dark aesthetic with intentional typographic choices. Automated deployment pipeline so changes ship without manual steps.
