CubeGoat

A Rubik's Cube solver that cracks any scramble in under 20 moves using the Kociemba algorithm, with 3D visualization and computer-vision color detection.

RoleFull-stack Developer
StackNext.js, FastAPI, Three.js, OpenCV

3D Cube Solver

Overview

CubeGoat solves any 3×3 Rubik's Cube configuration in 20 moves or fewer, guaranteed. The Kociemba Two-Phase algorithm runs on a FastAPI backend (hosted on Render), returning an optimal move sequence to the frontend where it's animated step-by-step on a Three.js WebGL cube.

The most interesting engineering challenge was the computer vision input mode: users can photograph a physical cube and the app extracts face colors via OpenCV with HSV normalization to handle variable lighting. This removes the tedious manual color-entry step entirely.

What I built

Three input methods — 3D paint mode, 2D grid entry, and CV image upload. Animated solution playback with 1×–5× speed control and step navigation. Full dark/light theme support. Next.js frontend on Vercel, FastAPI solver backend on Render, bridged via a REST API.
